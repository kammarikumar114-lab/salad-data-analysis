# 🥗 Salad Nutrition Analysis (SQL)

### 📊 Project Overview
This is a mini-project where I took a messy list of salad recipes and used SQL to clean the data and find the healthiest options. 

### 💡 Why I built this:
I wanted to practice **Data Cleaning** and **Window Functions**. In real life, data is never perfect, so I included a script to fix typos and missing calorie values.

### 🚀 Technical Skills Used:
* **Cleaning:** Using `UPDATE` and `WHERE` to fix data errors.
* **Aggregations:** `AVG` and `SUM` to find category totals.
* **Advanced SQL:** `WITH` (CTEs) and `RANK()` for deep analysis.

### 📈 Key Result:
The **Mediterranean Quinoa** salad was identified as the most balanced meal based on protein-to-calorie ratio.
